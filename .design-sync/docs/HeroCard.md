# HeroCard

A navy content card with a green eyebrow, headline, and body — a ready-made hero/feature block. Pass text via props (no children needed).

- `eyebrow` — small uppercase green kicker.
- `headline` — large white heading.
- `body` — muted supporting copy.

```jsx
<HeroCard
  eyebrow="Marine power"
  headline="Engineered for the open water"
  body="Reliable propulsion systems trusted across the fleet."
/>
```
