# Body

Paragraph body text, `<p>`. Three sizes; relaxed line-height for readability. Defaults to navy.

- `size`: `sm` | `base` | `lg`.

```jsx
<Body>Trusted propulsion systems for commercial and leisure marine.</Body>
<Body size="lg" color="rgba(255,255,255,0.7)">Built to run, always.</Body>
```
