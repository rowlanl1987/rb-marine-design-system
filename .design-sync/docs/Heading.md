# Heading

Section headings, `<h1>`–`<h4>` via `level`. Sizes step down 40 → 20px. Defaults to navy for light backgrounds.

- `level`: `1` | `2` | `3` | `4` (also sets the HTML tag).

```jsx
<Heading level={2}>Our fleet</Heading>
<Heading level={4} color="#ffffff">Specifications</Heading>
```
